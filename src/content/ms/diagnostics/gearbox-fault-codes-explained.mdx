Dongle OBD murah dah buat sesuatu yang menarik pada bidang kami: customer sekarang datang pegang code. Kadang membantu. Kadang ada abang dah beli dua solenoid dan satu wiring harness untuk masalah yang rupanya bateri RM 480. Bezanya faham apa fault code tu sebenarnya: kenyataan saksi, bukan hukuman.

Ini code yang paling banyak rakyat Malaysia search, diterjemahkan.

## P0700 — loceng pintu

Code paling disalahfaham dalam kamus OBD. P0700 maksudnya modul kawalan transmission dah simpan satu fault dan minta modul enjin pasang lampu warning. Itu je. Fault code sebenar duduk DALAM modul transmission, yang banyak scanner murah tak boleh masuk. Jadi dia tunjuk P0700, report "tak ada fault lain", dan semua orang keliru. Kalau scan anda berhenti kat P0700, scan tu belum habis.

## P0730, P0731-P0736 — ratio gear salah

Komputer arahkan satu ratio, terukur ratio lain. Terjemahan: ada yang slip (clutch pack, band, atau belt CVT) atau speed sensor tipu. Pada CVT, keluarga code ni layak dapat urgency, sebab [belt slip makan hardware ikut trip](/ms/cvt/cvt-gearbox-problems-symptoms). Level dan keadaan minyak dicheck dulu, sentiasa; separuh kes bermula situ.

## P0741 — clutch torque converter tak engage

Clutch lock-up dalam converter tak cengkam macam diarah. Rasa dia shudder masa cruise atau penggunaan minyak menaik (kuota [BUDI95](/ms/cvt/cvt-fuel-consumption-budi95) anda perasan walaupun anda tak). Punca dari minyak rosak sampai converter haus sampai solenoid. Code yang baik perangai, yang bagi ganjaran pada diagnosis betul dan menghukum tekaan part.

## P0715 / P0720 — litar speed sensor

Sensor kelajuan input dan output: mata gearbox sendiri. Bila satu gugur, shifting jadi tak menentu atau box kunci limp mode, dan pada sesetengah kereta (owner Mercedes 722.9 tahu) sensor tu duduk dalam, atas conductor plate. Wiring dan connector punca sebahagian besar kes ni, lagi-lagi [lepas air masuk campur tangan](/ms/maintenance/flooded-gearbox-what-to-do).

## P0768, siri P0973 — litar solenoid

Fault elektrik dalam litar solenoid shift. Perhatikan perkataan litar: solenoid tu sendiri, wiring dia, connector dia atau driver dalam control unit semuanya calon. Inilah keluarga code di mana pengukuran RM 90 kalahkan tekaan RM 900.

## P17xx dan kawan — dialek pengeluar sendiri

Code bermula P17-P18 tu khusus pengeluar, dan sinilah fault mechatronic DSG, counter gegar CVT dan adaptation ZF tinggal. Perlukan alat dan ilmu jenama untuk dibaca bermakna. Reader generik sama ada tak nampak atau tunjuk nombor tanpa cerita.

## Apa nak buat dengan code tu

Catat, gambar freeze-frame kalau app tunjuk, dan tahan diri dari shopping cart. Lepas tu dapatkan scan yang mampu baca modul transmission penuh: code plus live data plus adaptation plus sejarah suhu. Itulah [proses diagnosis](/ms/diagnostics/how-gearbox-diagnosis-works) dalam satu ayat, kosnya beberapa ratus ringgit, dan itulah beza antara fix RM 900 dengan musibah RM 5,000.

WhatsApp code dan model kereta anda ke [MNA Dynamic Torque kat Shah Alam](workshop:mna-dynamic-torque) atau, kat utara, [IM Dynamic Torque kat Simpang Ampat](workshop:im-dynamic-torque). Serius, hantar je code tu, memang berguna untuk triage. Mereka bagitau sama ada boleh pandu masuk, kena tow, atau relax boleh tunggu.

Code tu buka perbualan. Jangan biar sesiapa layan dia macam kesimpulan.
