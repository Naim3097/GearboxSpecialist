没有人会为波箱做预算。它是那种不请自来的维修，而且同一个症状，你可能在同一个下午拿到从 RM 300 到 RM 15,000 的报价。看了这么多年报价单是怎么写出来的，下面是 2026 年真实的价钱全景，让别人给你数字的时候，你心里先有一把尺。

## 价钱阶梯

| 项目 | 大概费用（RM） |
| --- | --- |
| 专业检测（路试 + 电脑扫描） | 100 – 300 |
| 换油 + 滤芯 | 250 – 600 |
| 换电磁阀（solenoid） | 800 – 2,500 |
| 阀体（valve body）维修或更换 | 1,500 – 4,000 |
| Torque converter（含拆装） | 1,800 – 3,500 |
| Mechatronic 维修（DSG / ZF） | 2,500 – 5,000 |
| 全套大修：国产车 4AT | 2,500 – 3,500 |
| 全套大修：日系 AT / CVT | 3,500 – 7,500 |
| 全套大修：DSG / DCT | 5,000 – 9,000 |
| 全套大修：欧陆 AT | 7,000 – 12,000+ |
| 原厂全新总成 | 15,000 – 40,000+ |

这张表有两点值得多看一眼。第一，现实中大部分波箱问题落在表的上半段，虽然某些店的报价习惯会让你以为凡事都五千起跳。第二，大修价和原厂新件价之间那道鸿沟，就是波箱专修这个行业存在的理由。

## 什么在决定价钱

**波箱本身，不是车标。** Proton X50 的 7DCT 重造起来比 Perodua 的 4AT 贵，是因为壳里面装的东西，不是引擎盖上的牌子。欧陆波箱更贵，因为零件按欧元计价，诊断软件也只有专门店才有。

**你来得多早。** 同一颗 JATCO CVT，异响刚出现时是 RM 2,000 的维修，拖六个月就是 RM 7,000 的大修。这篇文章里所有的变量，只有时机完全掌握在你手上。

**零件等级。** 原厂、优质副厂（常常同一条生产线出来，只是没打标）、还有夜市货。价钱差三倍，寿命差十倍。不知道用哪个等级的报价单毫无意义，所以开口问。

**价钱里包了什么。** 有的店大修价包 torque converter、冲洗散热管路、十二个月保修。有的一样都不包。旧的 converter 带着铁屑装回去，便宜就不是真的便宜。

## 各品牌速览

- **Perodua 和 Proton 4AT：** 市场上最亲民的大修，RM 2,500 到 3,500，零件到处有。
- **Proton Punch CVT：** 认真重造 RM 3,500 到 5,000，记得指定升级版 start clutch 零件。
- **Honda CVT：** 很多抖动案例 RM 500 以内就解决。Converter 工程 RM 2,000 到 3,500，全套大修 RM 4,500 到 7,000。
- **Toyota：** 故障多半跟油有关，所以维修常落在便宜那端。大修 RM 3,500 到 6,500。
- **Nissan（JATCO CVT）：** 最常走到全套重造那一步的家族。预算 RM 4,500 到 7,500，用升级零件。
- **VW 和 Audi DSG：** Mechatronic RM 2,500 到 5,000，离合器组 RM 3,000 到 5,500。DQ200 两样一起做，接近 RM 9,000。
- **BMW 和 Mercedes：** 很多人意外地发现问题停在电子或 mechatronic 层面就解决了，RM 1,500 到 5,000。全套大修 RM 7,000 到 12,000 以上，仍然不到代理厂换新价的一半。

## 让报价变诚实的五个问题

1. 到底坏了什么，证据在哪里？数字背后要有检测，不是耸耸肩。
2. 包哪些零件、什么等级、torque converter 算在里面吗？
3. 散热管路是冲洗还是更换？
4. 保修几个月、多少公里，写下来吗？
5. 拆开后发现别的问题，怎么报价、谁来批准？

五个问题都答得从容不迫的店，钥匙可以放心交给它。
